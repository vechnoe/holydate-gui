=====================
qjuliancalendarwidget
=====================

Description
===========

This package represents custom Qt QCalendarWidget for
Qt 4 version. Widget shown dates of Julian calendar.

Instalation
===========
For compilation need Qt 4.x, and qt-sources is required.

In package directory:

::
   
    qmake 
    make
    python configure.py
    make
    make install
